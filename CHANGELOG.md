feat(fu-273): TableComponent added Output event "rowClicked({$event: MouseEvent; rowIndex: string; rowEl: Element; rowData: any})"

- table-options.ts:  For suppress emit this event by click on some elements in row added property 'parentSelectorForRowClickStopPropagation: string[]' in interface TableOptions.rowsOptions.
- table.config.ts: Added list of parent selectors in case click event target - child of one of these selectors, event "rowClicked" will NOT be emitted
```
export const ROW_CLICK_SUPPRESS_FOR_PARENT_SELECTORS = [
    'fusion-checkbox',
    'fusion-toggle',
    'fusion-input',
    'fusion-icon.icon-edit',
    'fusion-icon.icon-save',
    'button' 
  ];
```
- table.component.ts added event emitter with argument - {$event: MouseEvent; rowIndex: string; rowEl: Element; rowData: any}.
  - $event: MouseEvent - event
  - rowIndex: string - row in table index. In table with total row rowIndex-"0" mean that click was on total row
  - rowEl: Element - native "tr" DOM Element
  - rowData: any - row data that was passed to table component in rows[] (by rowIndex)

    * changed all <div based menu items to <a link.
    * changes in "export interface SidebarMenuItem" for "routeConfigurations" property
        * deprecated SidebarMenuItem.routeConfigurations.href - use SidebarMenuItem.route instead.

    * All table data mast be pre-loaded (sync)
        * You do not need event (expandRow) listener all data already exist in the table [rows] input.
        * Input [(expandedRows)]="expandedRows" for expanded row indexes map now has two-way binding (if no event (expandRow) listener). So by changing it programmatically from host
          it will expand / collapse all (use **"default"** as map key) or one (use **row index** as map key) table row. Also it will be changed if
          some row was expanded / collapsed by table row component "expand" toggle icon click.
