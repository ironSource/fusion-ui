In case option [isBulkInsertTags]="true" when user type in search input or make paste some string with delimiter (TagsInputBulkInsertOptions->delimiter) (by default - comma ',') fusion-tags-input stopped emmit event 'searchChange' and by press 'Enter' key emmit 'bulkInsert' event with array of string for search.

This array created by split entered to the input string by TagsInputBulkInsertOptions->delimiter (default comma ','), filtered invalid by TagsInputBulkInsertOptions->validateKeyMethod (default blank values will be removed) and duplicated will be removed after validation filter.

In case 'bulk insert behaviour' dropdown will not be opened and consumer add it to the selected tags (by this.bulkInsertedTags.setValue(...NEW_TAGS_ARRAY...); on example)
