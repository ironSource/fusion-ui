# Color Palette

The palette allows you to modify the color of the components to suit your brand.

You can explore the default values of FusionUI color palette or adjust the colors by using CSS variables to fit you project theme.
<br/>
## Usage
Just put ```@import '@ironsource/fusion-ui/style/v3/vars/_colors';``` in your ***styles.scss***
Use as scss variable with prefix "$fu-...".

For example **$fu-dark-500**.
