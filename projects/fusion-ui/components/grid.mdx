# Grid system

The Grid layout is here to help you align blocks and widgets into columns and rows in a specific area.
By following a pattern made of containers with CSS classes, you can easily design your customized tab content, keeping a visual consistency, readability, and adaptability between screens.

To start using the grid, you need to import fusionUI grid to your project.
```css
    @import '~@ironsource/fusion-ui/style/grid.css';
```

### Grid gap
Our grid columns have a 16px padding on either side that results in a 32px gutter between columns and a 16px gutter on the grid edges. Even though our columns are fluid, the gutter remains constant.

`$gridColumnGap: 32px !default;`

### Offsetting columns
Move columns to the right using `.offset-*` classes. These classes increase the left margin of a column by * columns. For example, `.offset-4` moves `.col-4` over four columns.
