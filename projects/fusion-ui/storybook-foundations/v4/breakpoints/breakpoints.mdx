# Breakpoints

Use this set of standard breakpoints to maintain layout integrity across screen sizes. For best results, test designs and code at each of these standard breakpoints.

To start using breakpoints mixins, you need to import our `@import '~@ironsource/fusion-ui/style/v4/mixins/breakpoints` to your project and use mixins.

#### Using breakpoints mixins:
Media of at least the minimum breakpoint width. No query for the smallest breakpoint. Makes the @content apply to the given breakpoint and wider.

<Source language="css" code={`
@include media-breakpoint-min('sm') {
    .your-content{
        display: none
    }
}
`}/>

Media of at most the maximum breakpoint width. No query for the largest breakpoint. Makes the @content apply to the given breakpoint and narrower.

<Source language="css" code={`
@include media-breakpoint-min('sm') {
    .your-content{
        display: none
    }
}
`}/>

Media that spans multiple breakpoint widths. Makes the @content apply between the min and max breakpoints
<Source language="css" code={`
@include media-breakpoint-between('sm','md') {
    .your-content{
        display: none
    }
}
`}/>

No minimum for the smallest breakpoint, and no maximum for the largest one. Makes the @content apply only to the given breakpoint, not viewports any wider or narrower.
<Source language="css" code={`
@include media-breakpoint-only('sm') {
    .your-content{
        display: none
    }
}
`}/>
